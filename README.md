## 🚀 실행 방법
### 레포지토리 클론
```bash
$ git clone https://github.com/chloeeee210/mini-board.git
```
### 의존성 설치
```bash
$ npm install
```
### 프로젝트 실행
```bash
$ npm run dev
```
